# FinLit Quest 🪙
**India's Gamified AI Wealth Advisor for Gen-Z**  
Built for the **StoxraHack National Online Hackathon 2026** by Team **FinLits**.

## 🚀 Overview
FinLit Quest solves the financial illiteracy crisis among Indian youth by replacing traditional, boring text lectures with gamified byte-sized micro-quests (built with React Native) and a risk-free paper trading simulation arena powered by real-time Indian stock market feeds (`yfinance`). A 24/7 vernacular AI Companion (`Google Gemini API`) analyzes portfolio decisions to help students form secure wealth habits.

## 🛠️ Technical Stack
- **Frontend UI:** React Native, Expo Framework, Tailwind CSS
- **Backend API Core:** Python, FastAPI Server Architecture
- **Contextual Intelligence:** Google Gemini Pro API via LangChain
- **Market Data Engine:** yfinance Integration (NSE/BSE Feeds)
- **Database Layer:** Firebase Firestore

## 📂 Codebase Directory Structure
- `/frontend`: Cross-platform mobile layouts, custom card-swipe interfaces, and asset states.
- `/backend`: Scalable FastAPI endpoints handling market ingestion and conversational AI orchestration.
